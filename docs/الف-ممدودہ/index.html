الف ممدودہ

الف ممدودہ کا مسئلہ صرف مرکّبات میں پیدا ہوتا ہے۔ یعنی دل آرام لکھا جائے یا دلارام۔ ایسی صورت میں اصول یہ ہونا چاہیے کہ معیاری تلفّظ کو رہنما بنایا جائے اور مرکّب جیسے بولا جاتا ہو، ویسے ہی لکھا جائے۔
